Shows a popup of recent WooCommerce purchases with customer initials, country, product details, and time since purchase.

== Description ==

A simple, lightweight plugin that shows real-time notifications of recent WooCommerce purchases. The popup cycles through the latest 10 orders, showing customer initials, country (with flag), product name and image (both clickable), and how long ago the purchase was made.

- Displays purchases like: "J.M. from Italy bought"
- Clickable product image and name open in a new tab
- Shows time since purchase: Just now, minutes, hours, days, weeks, months
- Responsive, modern UI
- No settings: install and it just works

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/wc-just-bought directory or install directly from the Plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Ensure WooCommerce is installed and activated.
4. The popup will appear automatically on the frontend.

== Frequently Asked Questions ==

= Does this plugin have settings? =
No. This first version is intentionally simple. If you need customizations, contact info@shambix.com.

= What orders are shown? =
The latest 10 orders with status completed or processing.

= Is customer data protected? =
Yes. Only initials and country are shown. No sensitive data is exposed.
